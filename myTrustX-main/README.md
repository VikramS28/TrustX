# TrustX: Web Application for Transparent Transactions

TrustX is a web application designed to ensure **transparency and trust in digital transactions** using **blockchain technology**. The system securely records and verifies transaction data on the blockchain, making all records **tamper-proof and auditable**.

## Features

- Secure backend transaction handling using Python
- Blockchain-based validation and storage of transaction data
- Smart contracts for transparent and immutable records
- Integration with Ethereum / Polygon blockchain networks
- Secure user authentication and data storage
- Interactive web interface for transaction management

## Technologies Used

- **Backend:** Python, Flask  
- **Blockchain:** Ethereum / Polygon, Solidity  
- **Frontend:** Next.js  
- **Web3 Integration:** Web3.py  
- **Core Concepts:** Blockchain Transparency, Smart Contracts, Secure Transactions  

## How It Works

### Backend (Python & Flask)

- Handles transaction requests securely
- Manages user authentication and data storage
- Sends verified transaction data to the blockchain using Web3.py

### Blockchain Layer

- Smart contracts written in Solidity record transaction details
- Transactions are validated and stored on Ethereum/Polygon
- Ensures immutability and transparency of records

### Frontend

- Built using Next.js
- Allows users to initiate and view transactions
- Displays blockchain-verified transaction information

## Responsibilities

- Developed backend services using Python and Flask
- Implemented secure login and data management
- Integrated blockchain functionality with the backend using Web3.py

## Purpose

This project demonstrates how **blockchain technology** can be used to build **secure, transparent, and trustworthy web applications**.
